Background

QR77s is a high-end 77GHz 4D radar system developed by Qamcom Research and Technology AB for obstacle detection, area surveillance and multiple target tracking. This radar system was originally developed for obstacle detection in Schweizer Electronic AGs railway level crossing surveillance system FLEX-Ray 2.0.

FLEX-Ray 2.0

FLEX-Ray 2.0 is a SIL-3 failsafe automatic obstacle detection system for railway level crossings. The main function of this system is to determine if the area between the barriers is free from obstacles or not when the barriers are lowered. The FLEX-Ray system will signal that full-speed passage is allowed if each and every radar sensor in the system have found that this area is free of obstacles.

Other possible applications

The radar system QR77s can be adapted to a wide variety of foreseeable radar system applications. Here some of the foreseeable applications.

Stationary applications

  • Surveillance of soft targets and critical infrastructures

  • Surveillance of integrity sensitive areas where cameras cannot be used

Mobile applications

  • Obstacle detection and collision avoidance systems in mobile industrial, farming and forestry equipment

QRR77 Radar System Functionality

The radar system consists of the following components and functions.

Front-End

  • Microstrip patch antenna and radar chip integrated on same board

  • 2 Radar chips

  • Generates TX-signals at E-band.

  • Down conversion (demodulation) of received RF signals to baseband signals

Back-End

  • Analog pre-filtering and AD-conversion of baseband signals

  • Sampling of baseband signals up to 10-20 MHz

  • Controlling of front-end waveforms (e.g., start and stop of chirp, chirp rates, etc)

  • Distributing pre-filtered and downsampled data to the Signal Processing Unit

Signal Processing and tracking unit

  • Provides enhance (change) detection performance

  • Sophisticated multiple object tracking

  • GPU-based platform

  • Integrated in the back-end
